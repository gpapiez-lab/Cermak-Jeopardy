# Audio Resources

This directory should contain the following audio files for the Jeopardy game:

1. `jeopardy_theme.mp3` - The main Jeopardy theme music (background music)
2. `question_sound.mp3` - Sound effect played when a question is selected
3. `correct.mp3` - Sound effect played for correct answers
4. `incorrect.mp3` - Sound effect played for incorrect answers
5. `daily_double.mp3` - Sound effect played when a Daily Double is revealed
6. `final_jeopardy.mp3` - Sound effect played during Final Jeopardy

You can obtain these audio files from various sources:
- The Jeopardy theme music can be found in the public domain
- Sound effects can be created or downloaded from free sound effect websites
- The Daily Double and Final Jeopardy sounds are iconic to the show and can be found in various sound effect collections

Place all audio files in this directory and make sure they are added to your Xcode project target.

Note: Make sure you have the rights to use any audio files you include in your project.
